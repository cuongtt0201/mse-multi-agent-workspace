KIẾN TRÚC MSE — MASTER SPECIFICATION
  • Tách 2 mặt phẳng: RabbitMQ chở tín hiệu siêu nhẹ; Redis Blackboard (plan:{id}:context, HASH) giữ data — đa-pod ghi song song không lost-update.
  • Write-before-Publish: Agent ghi kết quả + completed lên Blackboard TRƯỚC khi báo downstream.
  • Worker FSM: classify → execute → reflect (nới query khi rỗng, ≤2 lần) → ask_user (HITL).
  • OAuth Reconnect: nâng scope (đọc→ghi) cần Disconnect/Connect lại; Gateway refresh & inject token vào platform_tokens.
